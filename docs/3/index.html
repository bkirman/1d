3. Making the Controller

As fun as Pressy Pressy Lighty Lighty is, it is annoying to use that tiny button, so let's add a new arcade style controller.

From the kit, you need that bundle of red and black wire, plus the two little red connectors.

Crimping

The wire has a connector at one end already, but we need to do the other one. The button we are using has "spade" type connectors so we need to crimp the matching socket onto our wire.

Using wire strippers, remove the last 5mm of plastic sheath from the end of the wire without the connector, to expose the wires underneath. Now, for each of the red and black wire, pop a red connector onto the bare wire, circle side first.

Check the connector is on the right way round - the "B" shape should be outwards. Using the crimping tool, line up the thinnest section of the connector with the red dot as shown below (NOT the cutter!!), squish it really hard! You should not be able to pull the wire out easily.

Assemble!

Now you will also need an Arcade button, and an button enclosure, from Ben. Choose your favourite combination of colours.

First step is to carefully feed the wire through the hole in the enclosure so that the red connectors are coming up through the middle. You might find it easier to feed in the black connector from the inside out.

Pull the wire through a bit and connect the red connectors to the flat metal spade connectors underneath the arcade button. The colour does not matter. Try to have it so the curves of the "B" connector are nearest the edge of the arcade button (i.e. facing down in the photo below).

Now, gently bend the assembled connection across the bottom of the button, inwardly. It won't get fully flat but that's ok. This is why the orientation in the last step is useful.

Gently feed the wire back through the hole in the enclosure, and sit the button on top.

Carefully line up the notches in the enclosure with the tabs on the button. Push the button into place.

It should fit very snugly. If you need to remove it, carefully use a screwdriver in the wire hole, to lever it back out.

You've made a controller! Now, plug the black connector at the other end of the wire onto the two pins on the bottom of the Plasma. The orientation and wire colour does not matter.

Plug the Plasma back into your computer if you disconnected it, run the game again, and enjoy the new clicky controller.

That's Super Pressy Pressy Lighty Lighty Champion Edition!

Things that might have gone wrong

If anything broke or snapped, we have a couple of spares. At worst we can quickly solder the wires for you.

If you finished it and the game is running, the button on the plasma works, but your button doesn't, probably there is a poor connection somewhere. Using a multimeter, you can do continuity tests to check the wire at both ends. We're happy to help with this!

Advanced

If you want to 3d print your own, the STL for the controller enclosure is here.

You could also think about other kinds of controllers. More buttons, knobs and sliders. Check out Line Wobbler by Robin Baumgarten - it uses a springy doorstop! Glow Golf by Alex Johansson uses a light sensor!
