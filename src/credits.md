This workshop has been written by [Ben Kirman](https://ben.kirman.org) for [EMF 2026](https://emfcamp.org). If you have any questions you can email me at ben@kirman.org.

## Code
The [complete code is available on the MIT licence at github](https://github.com/bkirman/1d_engine).

Feel free to fork and keep working with it. If you want to "reset" your Plasma, just download the contents of that respository and copy the file structure over to the Plasma board.

The [code for this website is also on github](https://github.com/bkirman/1d).

The workshop relies on the [Plasma](https://github.com/pimoroni/plasma) and [EZWifi](https://github.com/pimoroni/ezwifi-micropython) libraries from Pimoroni.
